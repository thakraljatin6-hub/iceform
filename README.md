# IceForm

Scientific Ice Cream Formulation Engine

IceForm is a web-based dairy formulation tool that automatically calculates ingredient quantities for ice cream mix based on target fat, SNF, sugar and total solids.

Features:
- Multi-ingredient formulation solver
- Automatic sugar adjustment
- Advanced guard system
- Quality insight analysis
- Batch scaling

Built using Python and Streamlit.

Live App: https://iceform.streamlit.app
